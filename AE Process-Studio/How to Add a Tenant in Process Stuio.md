__Add a new Tenant in AutomationEdge Process Studio__

AutomationEdge is a multi-tenant system. It keeps users, workflows, and other objects isolated so that one tenant cannot access another tenant’s data. In this chapter, you will learn how to add and delete tenants and understand the different tenant statuses.

__Prerequisites__

* Access to AutomationEdge Cloud Instance

To add a new tenant,
![Tenant Screen](..assets/Tenant-screen.png)
1. In the menu, click __Tenant Users__.
2. On the Tenants page, click __Add New__. 

![Tenant Page](..assets/Tenant-page.png)

3.  On the __Add New Tenant dialog__, perform the following:
   
![Tenant Dialog](..assets/Tenant-Dialog.png)

    a.  In the __Tenant Name__ field, enter a unique name with alpha-numeric characters.

    b. In the __Description__ field, enter the description of tennant.

    c. In the __Organization Code__, specify a unique organiation code. 

    d. The **File Based License** checkbox remains selected by default.
4. Click __Create__.
   
A new Tenant is added to your process studio, you can check the details on the Tenant page.
